# Visual Compiler Playground

AI Compiler와 AI Framework 내부에서 일어나는 일을
“보이는 형태”로 이해하기 위한 실험적 공간입니다.

이 폴더는 코드 설명이 아니라,
- Op가 어떻게 쌓이고
- Lowering이 무엇을 의미하며
- Kernel Selection이 언제, 왜 바뀌는지를
직관적으로 보여주는 것을 목표로 합니다.

Scratch처럼 블록을 쌓듯 연산을 구성하고,
실행과 함께 커널 선택의 변화를 관찰합니다.

This is not a tutorial.
This is a playground to *see how a compiler thinks*.
